# <a name="development-test-and-production-databases-visual-database-tools"></a>Entwicklungs-, Test- und Produktionsdatenbanken (Visual Database Tools)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] Wenn Sie zwei Datenbanken mit identischer Struktur besitzen, können Sie Änderungen an einer der beiden Datenbanken vornehmen und an die andere weitergeben. Wenn Sie z. B. über eine persönliche Entwicklungsdatenbank und eine für die Gruppe vorgesehene Testdatenbank verfügen, können Sie die Entwicklungsdatenbank ändern und die Änderungen dann an die Testdatenbank weitergeben.  
  
Führen Sie hierzu alle Änderungen in einer einzigen Sitzung mit der Entwicklungsdatenbank aus, erstellen Sie ein Änderungsskript der Sitzung, und führen Sie das Skript später für die Testdatenbank aus.
