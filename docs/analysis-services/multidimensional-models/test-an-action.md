# <a name="test-an-action"></a>Testen einer Aktion
[!INCLUDE[ssas-appliesto-sqlas](../../includes/ssas-appliesto-sqlas.md)]Zum Testen einer Aktion verwenden Sie die Browseransicht im Cube-Designer. Der Zugriff auf den Cube-Designer ist über [!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)]möglich. Nachdem Sie die Aktion erstellt haben, müssen Sie den Cube verarbeiten, bevor die Aktion getestet werden kann. Weitere Informationen finden Sie unter [Verarbeiten eines mehrdimensionalen Modells &#40;Analysis Services&#41;](../../analysis-services/multidimensional-models/processing-a-multidimensional-model-analysis-services.md).  
  
### <a name="to-test-an-action"></a>So testen Sie eine Aktion  
  
1.  Öffnen Sie den Cube, in dem die zu testende Aktion enthalten ist, und klicken Sie unter **Ansicht**auf **Browser**.  
  
2.  Ziehen Sie das Zielobjekt in **Spaltenfelder hierher ziehen** oder **Zeilenfelder hierher ziehen**.  
  
3.  Klicken Sie im Ansichtsbereich mit der rechten Maustaste auf das Objekt, an das die Aktion angefügt wird (z. B. eine Zelle). Der Aktionsname wird im Kontextmenü angezeigt.  
  
4.  Klicken Sie auf den Aktionsnamen, um die Aktion zu testen.
