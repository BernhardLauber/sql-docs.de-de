# <a name="cinstance-class"></a>CInstance-Klasse
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]Die [CInstance-Klasse](../../relational-databases/wmi-provider-configuration-classes/cinstance-class.md) stellt eine Instanz von einem [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Client.  
  
 Diese Klasse kann verwendet werden, um Protokolle und Netzwerkbibliotheken zu verwalten, die der Instanz eines [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] -Clients zugeordnet sind.
