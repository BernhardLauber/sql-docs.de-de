# <a name="device-not-ready-error"></a>Fehler 'Gerät nicht bereit'
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Diese Regel überprüft das Systemereignisprotokoll auf EventId 15. Dieser Fehler kann durch Konfigurationsprobleme beim SCSI-Hostadapter oder durch zugehörige Probleme verursacht werden.
