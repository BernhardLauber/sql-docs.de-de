# <a name="device-driver-control-error"></a>Gerätetreiber-Steuerungsfehler
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Diese Regel überprüft das Systemereignisprotokoll auf EventId 11. Dies könnte durch einen fehlerhaften Gerätetreiber, ein Hardwareproblem, eine fehlerhafte Verkabelung oder Konnektivitätsprobleme verursacht werden.
