# <a name="getresultsettype-method-sqlserverstatement"></a>GetResultSetType-Methode (SQLServerStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Ruft den Resultsettyp für [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) Objekte, die von diesem generierten [SQLServerStatement](../../../connect/jdbc/reference/sqlserverstatement-class.md) Objekt.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
public final int getResultSetType()  
```  
  
## <a name="return-value"></a>Rückgabewert  
 Ein **Int** , der den Typ des Resultsets angibt.  
  
## <a name="exceptions"></a>Ausnahmen  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Hinweise  
 Diese GetResultSetType-Methode wird von der GetResultSetType-Methode in der java.sql.Statement-Schnittstelle angegeben.
